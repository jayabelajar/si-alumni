# Phase 1: Project Setup

## Prompt 1.1: Environment Initialization
```text
Act as a Senior Backend Architect. Initialize a CodeIgniter 4 project for "SI Alumni".
1. Rename 'env' to '.env' and set CI_ENVIRONMENT to 'development'.
2. Set the 'app.baseURL' to your local development URL (e.g., http://localhost:8080).
3. Create a clean folder structure ensuring 'writable' directories have the correct permissions.
```

## Prompt 1.2: Database Configuration
```text
Continue as the Architect. Configure the database in '.env':
1. Set database.default.hostname to 'localhost'.
2. Set database.default.database to 'si_alumni'.
3. Set your local DB username and password.
4. Verify the connection by creating a simple migration test.
```
