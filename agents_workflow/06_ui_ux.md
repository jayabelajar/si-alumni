# Phase 6: Frontend & UI/UX

## Prompt 6.1: Main Layout System
```text
Act as a UI/UX Designer. Create a 'Views/layouts/main.php' file using Bootstrap 5:
1. Include a modern top-navbar and a sidebar for navigation.
2. Add sections for 'styles', 'content', and 'scripts'.
3. Ensure the design is responsive and uses a professional color palette (e.g., Deep Blue and Slate).
```

## Prompt 6.2: Views Integration
```text
Convert the Auth, Dashboard, and CRUD forms to use the 'main' layout:
1. Use 'section()' and 'extend()' CI4 view features.
2. Style tables with 'table-hover' and 'table-striped'.
3. Use Bootstrap 'cards' for dashboard stats.
```
