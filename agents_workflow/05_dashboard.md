# Phase 5: Dashboard & Search

## Prompt 5.1: Admin Dashboard Statistics
```text
Act as a Fullstack Developer. Create an 'AdminDashboard' controller:
1. Calculate counts for: Total Alumni, Total Employed, Total Unemployed.
2. Group alumni by graduation year for chart data.
3. Pass this data to a clean dashboard view.
```

## Prompt 5.2: Search & Advanced Filter
```text
Implement a Search function in the Admin Alumni list:
1. Add filters for 'Graduation Year', 'Major', and 'Employment Status'.
2. Use CI4 Query Builder (or Model->where()) to handle multiple optional filters dynamically.
```
